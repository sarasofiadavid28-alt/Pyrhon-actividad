# Documentación del Proyecto

## Ejercicios con estructuras `if`, `elif` y `else` en Python

**Autora:** Sara Sofia David David
**Lenguaje principal:** Python
**Entorno de desarrollo:** Visual Studio
**Interfaz adicional:** HTML

---

## Descripción

Este proyecto contiene una colección de ejercicios desarrollados en Python que hacen uso de estructuras de control condicional (`if`, `elif`, `else`).

El objetivo es practicar la toma de decisiones en programación mediante diferentes algoritmos que resuelven problemas básicos.

---

## Estructura del proyecto

El proyecto está organizado de la siguiente manera:

* 25 archivos en Python que contienen los diferentes algoritmos
* 1 archivo `index.html` como interfaz complementaria

**Total de archivos:** 26

---

## Cómo ejecutar el proyecto

1. Abrir el proyecto en Visual Studio o en un editor compatible con Python
2. Ubicar el archivo principal (`index.py`)
3. Ejecutar el programa con el siguiente comando:

```bash
python index.py
```

---

## Funcionalidad

El sistema permite ejecutar distintos algoritmos desde consola.
Cada algoritmo representa un ejercicio independiente basado en estructuras condicionales.

---

## Notas

* Se recomienda tener Python instalado (versión 3 o superior)
* El archivo `index.py` actúa como punto de entrada del programa
